# Chandler Superior Maintenance

Marketing website for Chandler Superior Maintenance — commercial cleaning and facility maintenance serving apartments, government facilities, and commercial businesses across Indiana.

## Stack

Static HTML, CSS, and vanilla JavaScript. No build step required.

## Local preview

```sh
python3 -m http.server 8765
```

Then open http://localhost:8765.

## Deployment

Deployed to Vercel. Any static host (Netlify, GitHub Pages, Cloudflare Pages) will also work — the project has no build step.
